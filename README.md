# Inson Dubois Wood — Studio

A dark, editorial, gallery-first site for a New York residential interior design studio.
Plain HTML / CSS / JS — **no build step**. Open `index.html` or deploy the folder as-is.

## Stack
- Static HTML/CSS/JS
- [Lenis](https://github.com/darkroomengineering/lenis) — smooth scroll (CDN)
- [GSAP + ScrollTrigger](https://gsap.com/) — hero/image parallax (CDN)
- Google Fonts — Cormorant Garamond + Inter

## Design
Warm near-black ground, cream type, brass accent (a nod to a fiber-optic puffer's
glow). Work leads; copy is intentionally minimal. Custom cursor, loader, clip-wipe
image reveals, velocity-reactive marquee. Respects `prefers-reduced-motion`.

## Run
Just open `index.html`. No server needed.
